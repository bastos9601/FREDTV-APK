## 🚀 Cómo Compilar (3 Pasos)

### 1. Instalar EAS CLI
```bash
npm install -g eas-cli
```

### 2. Iniciar Sesión
```bash
eas login
```

### 3. Compilar
```bash
cd iptv-app
eas build --platform android --profile release
```

**Listo.** Tu APK estará en 10-15 minutos.
