┌─ PASO 1: PREPARAR ─────────────────────────────────────────────────────────┐
│                                                                             │
│  1. Abre terminal/CMD                                                      │
│  2. Navega a la carpeta: cd iptv-app                                       │
│  3. Instala EAS CLI (si no lo tienes):                                     │
│     npm install -g eas-cli                                                 │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ PASO 2: INICIAR SESIÓN ──────────────────────────────────────────────────┐
│                                                                             │
│  Ejecuta: eas login                                                        │
│  Ingresa tus credenciales de Expo                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ PASO 3: COMPILAR APK ────────────────────────────────────────────────────┐
│                                                                             │
│  Ejecuta: eas build --platform android --profile release                   │
│                                                                             │
│  ⏱️  Tiempo estimado: 10-15 minutos                                        │
│  📊 Tamaño esperado: 60-80 MB                                              │
│  ⚡ Velocidad: 2-3x más rápido que versiones anteriores                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ PASO 4: DESCARGAR ───────────────────────────────────────────────────────┐
│                                                                             │
│  1. Ve a: https://expo.dev/builds                                          │
│  2. Busca tu compilación (debe estar en "Completed")                       │
│  3. Descarga el APK                                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ PASO 5: INSTALAR ────────────────────────────────────────────────────────┐
│                                                                             │
│  Opción A: Instalar manualmente                                            │
│  - Copia el APK a tu dispositivo Android                                   │
│  - Abre el archivo y selecciona "Instalar"                                 │
│                                                                             │
│  Opción B: Instalar con ADB                                                │
│  - adb install -r ruta/al/apk.apk                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
