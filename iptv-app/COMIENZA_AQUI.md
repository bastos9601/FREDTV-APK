## 🚀 Compilación Rápida (3 Pasos)

```bash
# Paso 1: Limpiar caché (importante si tienes error de Babel)
# Windows: LIMPIAR_CACHE.bat
# Linux/Mac: bash limpiar-cache.sh

# Paso 2: Instalar EAS CLI
npm install -g eas-cli

# Paso 3: Iniciar sesión
eas login

# Paso 4: Compilar
eas build --platform android --profile release
```

**Listo.** Tu APK estará en 10-15 minutos.
